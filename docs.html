Quick Start

One-Command Installation

$ curl -fsSL https://neugi.sh/install | bash

Or manually:

$ git clone https://github.com/atharia-agi/neugi_swarm.git
$ cd neugi_swarm
$ python3 neugi_wizard.py

Installation

Requirements

  • Python 3.8+
  • Ollama (installed automatically)
  • Windows, macOS, or Linux

Installation Steps

Step 1: Clone Repository

$ git clone https://github.com/atharia-agi/neugi_swarm.git
$ cd neugi_swarm

Step 2: Run Wizard

$ python3 neugi_wizard.py

The wizard will guide you through:

  • Check Ollama installation
  • Auto-start Ollama if needed
  • Configure your name
  • Set up Telegram (optional)

Step 3: Start NEUGI

$ python3 neugi_swarm.py

Then open: http://localhost:19888

Troubleshooting

Ollama Not Running

$ ollama serve

Or let NEUGI start it automatically by running the wizard.

Port Already in Use

$ lsof -i :19888  # Find process
$ kill <PID>    # Kill it

Reset Configuration

$ rm ~/neugi/data/config.json
$ python3 neugi_wizard.py

Check System Status

$ curl http://localhost:19888/health

Security

NEUGI has built-in security with two modes:

Sandbox Mode (Default)

Secure by default! Blocks dangerous commands.

  • Blocks: rm -rf, sudo, shell injection
  • Allowed: python, git, pip (configurable)

Full Access Mode

For advanced users who need full control.

Warning

Full access allows executing ANY command. Only enable when you trust what you're doing!

$ python3 neugi_security.py
# Select option 2 and type 'YES' to confirm
