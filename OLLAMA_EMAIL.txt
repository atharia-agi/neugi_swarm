## About NEUGI

NEUGI is a user-friendly AI assistant that makes AI accessible to everyone:

- **One-command install**: `curl -fsSL https://raw.githubusercontent.com/atharia-agi/neugi_swarm/main/install.sh | bash`
- **Auto-setup wizard**: Guides users through configuration
- **Technician mode**: Built-in system doctor that fixes issues automatically
- **Dashboard**: Clean web UI for interaction
- **CLI**: Simple commands like `neugi start`, `neugi status`, `neugi technician`

## Key Features

- Powered by Ollama (qwen2.5:1.5b and local models)
- Works with free tier!
- Error recovery with auto Technician
- For beginners and advanced users alike
- Open source: https://github.com/atharia-agi/neugi_swarm
