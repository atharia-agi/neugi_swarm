## What is Neugi?

**Neugi Army** is an advanced autonomous multi-agent AI system designed to handle complex tasks through collaboration, self-evolution, and intelligent delegation.

## Features

- 🐝 **Swarm Intelligence** - Multiple agents collaborate autonomously
- 🧬 **Self-Evolution** - Agents improve themselves over time
- 💰 **Task Marketplace** - Agents earn tokens for completed tasks
- 🏪 **Agent Marketplace** - Buy/sell custom agent templates
- 📈 **Auto-Scaling** - Automatic resource management
- 📊 **Advanced Analytics** - Comprehensive dashboard

### Core Features (v11 Ultimate)

- Multi-channel support (Telegram, Discord, WhatsApp, etc)
- Voice (TTS + STT)
- Long-term memory
- Multiple LLM providers (Claude, GPT-4, MiniMax, Ollama)
- Gateway API server

## Quick Start

```bash
# One-line install
curl -sSL https://raw.githubusercontent.com/atharia-agi/neugi_army/main/neugi_army_v11.py -o neugi.py && python3 neugi.py
```

Or clone and run:

```bash
git clone https://github.com/atharia-agi/neugi_army.git
cd neugi_army
python3 neugi_army_v11.py
```

## Configuration

Copy `config_template.py` to `config.py` and add your API key:

```python
API_KEY = "your-api-key-here"
```
